NextUI (formerly MinUI Next)

A CFW based on MinUI with a rebuilt emulation engine and tons of added features for the TrimUI Brick and Smart Pro. For other devices, we recommend checking out MinUI.

Features:

  • Fixed screen tearing and sync stutter by rebuilding the emulator engine core
  • Game switcher menu (Onion OS style) by @frysee
  • High audio quality with libsamplerate
  • Much lower latency (average 20ms, 1 frame of 60fps)
  • Game art/media support
  • Game time tracker
  • Custom boot logos by @SolvalouArt
  • Color and font customization
  • Menu animations
  • LED control and ambient LED mode
  • Screen color temperature and brightness controls
  • Deep Sleep mode with instant ON and overheating fix
  • Battery Monitoring with history graph
  • Scrolling animated titles for long game names
  • Updated and optimized build emulation cores
  • FBNeo Arcade screen rotation
  • CJK font support for JP/CN rom names
  • Many smaller fixes and optimizations

Current Supported Devices:

  • Trimui Brick | MinUI-20241028-0
  • Trimui Smart Pro | MinUI-20231111b-2

Installing:

Copy MinUI.zip (do not unzip) and trimui to the root of your SD Card. Boot up your Trim UI device, and it will install/update automatically.

How to Use:

While in Menu:

  • Hold Start + Volume Up/Down to adjust brightness
  • Hold Select + Volume Up/Down to adjust color temperature
  • Press Select to open the game switcher menu

While in Game:

  • Press Menu to open in-game options
  • Hold Menu + Select to open the game switcher

Deep Sleep Mode:

First, the device enters light sleep (screen off, LEDs pulse 5 times). After 2 minutes, it enters full deep sleep (LEDs turn off).
